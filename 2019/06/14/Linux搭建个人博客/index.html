Linux搭建个人博客

2019年6月14日

Linux系统部署Hexo个人博客到Github

1.从官网下载nodejs

这里用的是10.15.3 LTS

官网地址

解压下载的压缩包
tar -xvf node-v10.15.3-linux-x64.tar.xz
建立软链接
ln -s /root/node-v10.15.3-linux-x64/bin/npm /usr/bin/npm
ln -s /root/node-v10.15.3-linux-x64/bin/node /usr/local/bin/node

在linux下输入node -v和npm -v看是否显示版本号,如果有显示则上一步成功

2.安装hexo,并发布本地博客

npm install -g hexo-cli

等待一段时间(比较慢),安装完毕后,建立软链接

ln -s /root/node-v10.15.3-linux-x64/bin/hexo /usr/bin/hexo

这样就安装完了.
接下来开始发布博客:
新建一个文件夹blog(名字自己起),cd命令进入目录后执行

hexo init

这样,hexo会帮你在blog目录下生产所需要的各种文件
然后执行下面两条命令,发布本地博客

hexo g
hexo s

然后ctrl+c结束

3.部署到Github

  • Github操作

    1. 注册Github账号
    2. 新建一个仓库（头像旁的加号New repository），仓库名为你的昵称.github.io
    3. 完成后回到Linux

  • Linux操作

    1. 在blog文件中修改配置文件_config.yml

    2. 在最下面的的参数

      # Deployment
## Docs: https://hexo.io/docs/deployment.html
deploy:
  type: git
  repo: https://github.com/你的昵称/你的昵称.github.io.git
  branch: master

    3. 安装deploy-git

      npm install hexo-deployer-git --save

    4. 安装完后把博客部署到GitHub

      hexo clean
hexo g
hexo d

    5. 最后一条命令输入完后,要输入github的用户名和密码

    6. 过一会你就可以在http://你的昵称.github.io这个网站看到你的博客了！！
